# 🍰 JOELi's DEVINE CAKERY - Website

A stunning, luxury patisserie website built with React, Tailwind CSS, and Framer Motion.

## ✨ Features
- **Animated Hero**: High-impact entry with floating sparkle particles.
- **Filterable Gallery**: Smoothly transition between cake categories using Framer Motion.
- **Custom Order System**: Fully validated order form with success states.
- **Luxury Aesthetic**: Cream, Gold, and Chocolate palette with Playfair Display and Dancing Script fonts.
- **Mobile Responsive**: Optimized for all screen sizes.
- **WhatsApp Integration**: Instant connection for custom inquiries.

## 🚀 Deployment Instructions (Beginner Friendly)

### 1. Push to GitHub
1. Create a new repository on GitHub.
2. In your terminal, run:
   ```bash
   git init
   git add .
   git commit -m "Initial commit - JOELi's Devine Cakery"
   git remote add origin <your-github-repo-url>
   git push -u origin main
   ```

### 2. Deploy to Netlify
1. Log in to [Netlify](https://www.netlify.com/).
2. Click **"Add new site"** $\rightarrow$ **"Import from an existing project"**.
3. Select **GitHub** and find your `joelis-cakery` repository.
4. Use the following build settings:
   - **Build Command**: `npm run build`
   - **Publish Directory**: `dist`
5. Click **"Deploy Site"**.

Your site is now live! 🎂✨

## 🛠 Tech Stack
